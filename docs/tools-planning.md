# toolBench 工具规划文档

## 项目概述

toolBench 是一款面向开发者的轻量级桌面工具箱，专注于解决日常开发中高频但繁琐的小问题。

**核心理念**：小而美、开箱即用、无需联网、可扩展。

---

## 已规划功能

### 端口管理（v0.1）
- [x] 端口占用查看
- [x] 端口占用清除

## 系统级工具痛点与解决方案

### 一、进程管理类

#### 1. 进程查看器
| 项目 | 内容 |
|------|------|
| 功能 | 展示所有运行中的进程、CPU/内存占用 |
| 痛点解决 | 找哪个进程占满了内存/CPU |
| 使用频率 | 高 |

#### 2. 进程杀手
| 项目 | 内容 |
|------|------|
| 功能 | 一键强制结束指定进程 |
| 痛点解决 | 某程序卡死无法关闭 |
| 使用频率 | 高 |

#### 3. 子进程树
| 项目 | 内容 |
|------|------|
| 功能 | 查看进程及其所有子进程 |
| 痛点解决 | 结束父进程时漏掉子进程 |
| 使用频率 | 中 |

## 功能优先级

| 优先级 | 工具 | 说明 |
|--------|------|------|
| P0 | 进程查看器 + 进程杀手 | 核心功能，与端口管理搭配 |
| P1 | 系统概览 + 大文件查找 | 高频使用 |
| P2 | 环境变量 + DNS 解析 | 实用工具 |
| P3 | 子进程树 + 连接追踪 | 进阶功能 |
| P3 | Hosts 管理 + 文件差异 | 辅助功能 |
